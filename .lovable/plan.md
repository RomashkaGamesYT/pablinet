

## План: Соцсеть «нэт» — полная реализация в React + Supabase

### Интерфейс (точная копия вашего HTML)
- **Левая навигация**: Лента, Поиск, Ивент, Уведомления, Профиль, Выйти — с теми же иконками (Iconify Solar), стилями и анимациями
- **Тёмная тема**: фон `#0e0e0f`, карточки `#141415`, кольца `ring-white/5` — всё как в оригинале
- **Шрифты**: Inter + Montserrat для логотипа «нэт»
- **Мобильная версия**: нижняя панель навигации
- **Табы-вьюшки**: переключение через React Router (/, /search, /events, /notifications, /profile)

### Авторизация (Supabase Auth)
- Страницы входа и регистрации в стиле приложения (тёмная тема, те же элементы)
- Сброс пароля с отдельной страницей `/reset-password`
- Защищённые маршруты — редирект на логин если не авторизован

### База данных (Supabase)
- **profiles** — имя, юзернейм, аватар-эмодзи, био, дата регистрации
- **posts** — текст, изображения, автор, таймстемпы
- **likes** — связь пользователь-пост
- **follows** — подписчики/подписки
- **notifications** — лайки, подписки, упоминания
- **events** — ивенты с регистрацией участников
- RLS-политики на все таблицы

### Функциональность
- **Лента**: создание постов, лайки, счётчик комментариев, шаринг — всё inline как в вашем дизайне
- **Поиск**: поиск по пользователям и постам, популярные теги
- **Профиль**: баннер, аватар с эмодзи, статистика подписчиков/подписок, редактирование профиля
- **Уведомления**: реальные уведомления о лайках и подписках
- **Ивенты**: список событий, кнопка «Участвовать»
